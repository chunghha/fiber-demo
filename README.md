## Run locally

```bash
go run main.go
```

## Run on local Docker

```bash
sh run.sh
```

## Swagger

```bash
go get -u github.com/swaggo/swag/cmd/swag
swag init
```

open [swagger](http://localhost:3000/swagger/index.html)
